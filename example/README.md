# 臺中工務段人事室專書閱讀區

一個專業的圖書館管理系統網頁，使用純 HTML、CSS 和 JavaScript 開發。

## 功能特色

### 📚 三大主要功能分頁

1. **115年度選書** - 展示本年度精選書籍，包含詳細的書籍資訊和內容簡介
2. **歷年書目** - 瀏覽所有歷年書籍，支援搜尋功能
3. **我的借閱** - 管理個人借閱記錄，包含目前借閱和歷史記錄

### ✨ 設計特點

- **現代化設計**：採用漸層背景、陰影效果和流暢動畫
- **響應式佈局**：完美支援桌面、平板和手機裝置
- **互動體驗**：
  - 平滑的分頁切換動畫
  - 書籍卡片懸停效果
  - 搜尋功能與即時過濾
  - 鍵盤快捷鍵支援（Ctrl + ← → 切換分頁）
- **專業配色**：使用藍色系主色調，搭配暖色系強調色
- **微互動**：按鈕點擊回饋、載入動畫等細節處理

### 🎨 視覺設計

- 使用 Google Fonts 的 Noto Sans TC 字體
- 漸層背景和玻璃擬態效果
- 卡片式設計語言
- 柔和的陰影和圓角
- 流暢的過渡動畫

## 使用方式

### 直接開啟

1. 下載或複製所有檔案到本地目錄
2. 使用瀏覽器開啟 `index.html` 檔案
3. 開始使用！

### 本地伺服器（建議）

使用 Python 快速啟動本地伺服器：

```bash
# Python 3
python -m http.server 8000

# 或使用 Python 2
python -m SimpleHTTPServer 8000
```

然後在瀏覽器中訪問 `http://localhost:8000`

### 使用 VS Code Live Server

1. 安裝 Live Server 擴充功能
2. 右鍵點擊 `index.html`
3. 選擇 "Open with Live Server"

## 功能說明

### 年度選書

- 展示本年度精選書籍
- 每本書包含：
  - 書籍封面圖片
  - 書名、作者、出版社、出版日期
  - 詳細內容簡介
  - 借閱按鈕（可用/已借出狀態）

### 歷年書目

- 網格式展示所有書籍
- 搜尋功能：
  - 支援書名、作者、關鍵字搜尋
  - 即時過濾結果
  - 無結果時顯示友善提示
- 卡片懸停效果與動畫

### 我的借閱

- **借閱規則說明**：清楚列出借閱規定
- **目前借閱記錄**：
  - 顯示借閱狀態（已逾期、即將到期、已歸還）
  - 支援續借功能
  - 撰寫讀書心得
- **歷史借閱記錄**：表格式展示過往借閱紀錄

## 互動功能

### 鍵盤快捷鍵

- `Ctrl + ←`：切換到上一個分頁
- `Ctrl + →`：切換到下一個分頁
- `Enter`：在搜尋框中執行搜尋

### 按鈕功能

- **我要借閱**：點擊後確認借閱，顯示處理動畫
- **續借**：延長借閱期限
- **撰寫讀書心得**：開啟心得撰寫功能（開發中）
- **搜尋**：過濾書籍列表

## 自訂設定

### 修改顏色主題

在 `styles.css` 的 `:root` 區塊中修改 CSS 變數：

```css
:root {
    --primary-color: #2c5f7c;    /* 主色調 */
    --accent-color: #ff6b6b;     /* 強調色 */
    /* ... 其他顏色設定 */
}
```

### 新增書籍

在 `index.html` 中複製書籍項目的 HTML 結構，並修改內容：

```html
<div class="book-card">
    <div class="book-card-cover">
        <img src="images/your-book.jpg" alt="書名">
    </div>
    <div class="book-card-info">
        <h4>書名：您的書名</h4>
        <p>作者：作者名稱</p>
        <p>出版社：出版社名稱</p>
        <p>出版年：2024</p>
        <button class="btn-borrow">我要借閱</button>
    </div>
</div>
```
